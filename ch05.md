# 4×4薄膜鍵盤
## 原理說明
薄膜按鍵模組（hex keypad），有4×4或3×4（少了最右邊一行的A, B, C, D鍵），</br>
由16個按鍵（開關）交織而成。有些按鍵模組直接使用按鍵（微觸）開關組裝</br>
<img width="600" height="300" alt="image" src="https://github.com/user-attachments/assets/fcc1cd15-4ac4-42d6-83d6-414e3418f49c" /></br>
4×4按鍵模組有8個接腳，分成列、行兩組，可以接在微控器的任意8個接腳</br>

### 按鍵偵測與掃描原理
把4×4按鍵簡化成3×1，像下圖這樣串連三個開關，連接到同一個微控制器的輸入腳。</br>
<img width="600" height="200" alt="image" src="https://github.com/user-attachments/assets/d232603a-f0b6-4600-98a9-ceec758083a2" /></br>

假設開關的「行1」到「行3」輸入端全都輸入高電位，無論開關是否被按下，Arduino將接收到高電位（1）。</br>
為了檢測到其中按鍵被按下，程式必須依序將「行1」~「行3」腳位設定成低電位。</br>
<img width="600" height="200" alt="image" src="https://github.com/user-attachments/assets/d100958e-8100-460a-b3cb-06aaf8b0b6d5" /></br>

輪到「行2」腳輸入低電位，此時，微控器的輸入腳也將接收到低電位（0），</br>
由此可知連接「行2」的「開關B」被按下了。</br>
<img width="600" height="200" alt="image" src="https://github.com/user-attachments/assets/de47be06-2fca-4402-946e-ab23c5ec229f" /></br>

輪到「行3」腳輸入低電位，由於「開關C」未被按下，</br>
因此微控器的輸入腳接收到高電位（1）。</br>
<img width="600" height="200" alt="image" src="https://github.com/user-attachments/assets/3ee01df2-8791-4cd5-8a8c-8367dcd8d020" /></br>

到此，偵測按鍵的程式必須再次回到「行1」，輸入低電位…</br>
如此反覆循環掃描，才能持續偵測到某個按鍵是否被按下。</br>
實際的程式需要運用雙重迴圈，才能分批掃描每一列：</br>
<img width="664" height="359" alt="image" src="https://github.com/user-attachments/assets/e0de6714-4397-43af-b263-97e896decdf5" /></br>

## 範例
```
//ch05_01
#include <Keypad.h>         // 鍵盤函式庫

const byte ROWS = 4;        // 宣告列為4，const為不能被更改，byte數值範圍0~255，占用1 byte記憶體
const byte COLS = 4;        // 宣告行為4，const為不能被更改，byte數值範圍0~255，占用1 byte記憶體
char keys[ROWS][COLS] = {   // 鍵盤對應字元
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};
byte rowPins[ROWS] = {0,4,16,17};   // row接腳
byte colPins[COLS] = {5,18,19,21};  // col接腳
Keypad keynum = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS );  // 創建Keypad物件叫做keynum，並放入上述的資料

void setup(){
  Serial.begin(115200);
}

void loop(){
  char key = keynum.getKey();        // 讀取按鍵值並存入key
  if (key != NO_KEY){                // 判斷是否有按鍵值
    Serial.println(key);             // 在序列埠監控視窗顯示按鍵值
  }
}
```

# 4×4薄膜鍵盤 + TrafficLight
## 程式整合
### TrafficLight(ch03_01)
```
//ch03_01
#define LED_R 12         // 將LED_R 定義為GPIO 12
#define LED_Y 14         // 將LED_Y 定義為GPIO 14
#define LED_G 27         // 將LED_G 定義為GPIO 27

void setup()
{
  // 初始化所有 LED 接腳為輸出模式
  pinMode(LED_R, OUTPUT);
  pinMode(LED_Y, OUTPUT);
  pinMode(LED_G, OUTPUT);
}

void loop()
{
  // 1. 紅燈亮 1 秒
  digitalWrite(LED_R, HIGH);
  digitalWrite(LED_Y, LOW);
  digitalWrite(LED_G, LOW);
  delay(1000);

  // 2. 黃燈亮 1 秒
  digitalWrite(LED_R, LOW);
  digitalWrite(LED_Y, HIGH);
  digitalWrite(LED_G, LOW);
  delay(1000);

  // 3. 綠燈亮 1 秒
  digitalWrite(LED_R, LOW);
  digitalWrite(LED_Y, LOW);
  digitalWrite(LED_G, HIGH);
  delay(1000);
}
```
