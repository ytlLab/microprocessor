# 可變電阻(電位器、Potentiometer)
<img width="600" height="400" alt="image" src="https://github.com/user-attachments/assets/06127120-6148-4331-a00e-6868d32a5e53" /></br>

（英文：Potentiometer，通俗上也簡稱 Pot，少數直譯成電位計），中文通常又稱為可變電阻器（VR，Variable Resistor）或簡稱可變電阻，</br>
是種具有三個端子，其中有兩個固定接點與一個滑動接點，可經由滑動而改變滑動端與兩個固定端間電阻值的電子零件，</br>
屬於被動元件，使用時可形成不同的分壓比率，改變滑動點的電位，因而得名。</br>

### 課程範例 讀取類比訊號
```
//ch06_01
# define pot 39            // 將Pot定義為GPIO 39
int val;                   // 存放可變電阻值的變數
void setup()
{
  Serial.begin(115200);
}
void loop()
{
  val = analogRead(pot);   // 讀取可變電阻值(0~4095) 
  Serial.println(val);
  delay(10);
}

```
analogRead(pin) 用於讀取輸入的電壓值，透過類比數位轉換器(ADC)將其轉換為數位值，</br>
支援12-bit，故此數值範圍為0~4095 (2¹² = 4096) </br>

## PWM(脈衝寬度調變)
- 目的：將數位輸出模擬成類比訊號
- 用途：調整燈光的亮度、馬達的轉速、RGB LED 的配色、螢幕亮度控制、喇叭的大小聲/聲音頻率

[剖析PWM脈寬調製](https://kknews.cc/zh-tw/news/xzqzeeq.html)</br>

<a href="https://www.youtube.com/watch?v=8tq9XV6g5bE" target="_blank">
    <img width="600" height="400" alt="image" src="https://github.com/user-attachments/assets/86e16be8-6e02-425b-a0f9-58c33d1cff88" />
</a>
analogWrite(pin, value) 用於產生 PWM訊號（脈衝寬度調變）</br>
支援8-bit，故此數值範圍為0~255 (2⁸= 256) </br>

- analogWrite(pin,0)   → 0% 佔空比（0V）
- analogWrite(pin,127) → 約 50% 佔空比（平均約 1.65V）
- analogWrite(pin,255) → 100% 佔空比（3.3V）

### 課程範例 呼吸燈
```
//ch06_02

# define LED 13            // 將LED定義為GPIO 13

void setup()
{
  pinMode(LED,OUTPUT);     //  LED腳位，設定為OUTPUT
}
void loop()
{
  for (int i =0; i<=255;i++){
    analogWrite(LED,i);          //  i從0~255
    delay(10);
  }
  for (int i =255; i>=0;i--){
    analogWrite(LED,i);          //  i從255~0
    delay(10);
  }
}
```

                                // 設定游標位置至(6,1)第二行第7格開始
                                // 顯示亮度比例
                                // 設定游標位置至(7,1)第二行第8格開始

  delay(10);
