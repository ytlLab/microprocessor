# 課程大綱
## 課程分為三個主要階段，從基礎模擬到實體應用與整合：

物聯網架構：
- 感知層：透過各種感測器蒐集數據（如溫度、濕度、二氧化碳、PM2.5）。
- 網路層：利用網路協定（HTTP、MQTT）將資料以 CSV 或 JSON 格式傳輸至伺服器。
- 應用層：進行資料分析（如空汙分布區域分析）並將結果可視化。

# 程式撰寫 (C++)
Arduino 程式主要由兩個核心函式組成：
void setup()：開機時只運作一次，用於設定腳位模式。
void loop()：重複執行直到關機為止。

<img width="300" height="400" alt="image" src="https://github.com/user-attachments/assets/3d9518d2-f20e-44ca-8322-be6e583bb3e1" />

C++

``` 
void setup() {
  pinMode(2, OUTPUT);    // 設定第 2 腳位為輸出
}

void loop() {
  digitalWrite(2, HIGH); // 第 2 腳位輸出高電位 (亮)
  delay(1000);           // 等待 1 秒
  digitalWrite(2, LOW);  // 第 2 腳位輸出低電位 (暗)
  delay(1000);           // 等待 1 秒
}
```
