# ESP32
ESP32 是一款由樂鑫科技（Espressif Systems）開發的低成本、低功耗的 單晶片系統（SoC）。它整合了 Wi-Fi 與雙模藍牙（傳統藍牙與低功耗藍牙 BLE），是目前物聯網（IoT）開發者、創客與工程師最受歡迎的選擇之一。

如果你曾經用過 Arduino，你可以把 ESP32 想像成其「性能大幅進化版」，不僅速度更快，還內建了無線通訊能力。

## 核心規格與特點
ESP32 的設計核心是為了提供極高的整合度與性能。以下是其主要的硬體規格：
| 特性 | 規格說明 |
| :--- | :--- |
| **處理器** | Xtensa® 雙核 32 位元 LX6 微處理器（也有單核版本） |
| **時脈頻率** | 可達 240 MHz |
| **無線連接** | Wi-Fi (802.11 b/g/n) 與 藍牙 (v4.2 BR/EDR 及 BLE) |
| **記憶體** | 520 KB 內部 SRAM，通常搭配 4MB 的外部 Flash 記憶體 |
| **豐富介面** | 高達 34 個 GPIO、ADC、DAC、SPI、I2C、UART、PWM、I2S |
| **感測器** | 內建電容式觸控感測器、霍爾感測器、溫度感測器 |
| **功耗** | 支援多種睡眠模式，深度睡眠 (Deep Sleep) 下電流僅需數微安培 |

## 優勢
1. 強大的通訊能力
   與前一代著名的 ESP8266 相比，ESP32 增加了 藍牙 功能，這讓它在與手機配對或建構藍牙網狀網路（Mesh Network）時展現出巨大的彈性。
2. 多核心運作
   大多數 ESP32 型號擁有 雙核心。這意味著你可以讓一個核心處理 Wi-Fi 連線與網路通訊，另一個核心則專注於讀取感測器數據或控制馬達，避免程式互相干擾導致系統崩潰。
3. 開源社群與生態系
   ESP32 支援多種開發環境，門檻極低：
    Arduino IDE: 最適合初學者。
    MicroPython: 使用 Python 語法開發，開發效率極高。
    ESP-IDF: 官方提供的開發框架，適合商業級產品開發。

## ESP32 的家族成員
隨著技術發展，樂鑫推出了不同的子系列來滿足特定需求：
- ESP32 (經典款): 雙核，功能最全面，支援傳統藍牙。
- ESP32-S 系列 (如 S2, S3): S3 具備 AI 加速指令，適合語音辨識或簡單影像處理。
- ESP32-C 系列 (如 C3, C6): 採用 RISC-V 架構，旨在取代 ESP8266，主打高性價比。
- ESP32-H 系列: 專注於 Matter、Thread 與 Zigbee 等物聯網通訊協定。
