# 課程大綱
##課程分為三個主要階段，從基礎模擬到實體應用與整合：
- 模擬電路：電路繪製、熟悉語法。
- 組裝模組：認識模組功能、訓練程式彙整、實體接線。
- 生活應用：日常問題發想、整合模組功能、報告撰寫。

物聯網架構：
- 感知層：透過各種感測器蒐集數據（如溫度、濕度、二氧化碳、PM2.5）。
- 網路層：利用網路協定（HTTP、MQTT）將資料以 CSV 或 JSON 格式傳輸至伺服器。
- 應用層：進行資料分析（如空汙分布區域分析）並將結果可視化。

<img width="400" height="300" alt="image" src="https://github.com/user-attachments/assets/729097e9-4ade-4493-ab98-9641dbde1cdd" />

硬體邏輯：如同人類大腦，微處理器（如 Arduino）接收來自輸入裝置（感測器）的訊號，處理後傳送指令給輸出裝置（LED、馬達）。

# 麵包板 (Breadboard)
結構：匯流排 (Power Rails)：兩側紅藍線處，縱向每 25 點導通。
內部插槽：橫向（a-b-c-d-e 為一組，f-g-h-i-j 為一組）每 5 點導通。
